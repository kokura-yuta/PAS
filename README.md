# PAS - Personal AI System

PASは、一人ひとりを長期的に理解し、人生に寄り添うことを目指したAIアシスタントです。

## 現在できること

- FastAPIでWebアプリを起動
- ホーム画面の表示
- チャット画面の表示
- OpenAI APIを使ったAIチャット
- PostgreSQLへの会話保存
- 直近の会話履歴を使った返答

## 使用技術

- Python
- FastAPI
- HTML
- Jinja2
- OpenAI API
- SQLAlchemy
- PostgreSQL
